# Handwritten Text Recognition (HTR) — Распознавание русской рукописи

Веб-приложение на базе нейросетей для автоматического обнаружения, распознавания и исправления ошибок в русском рукописном тексте.

## Основные возможности

* **Детекция слов (DocTR)**: Сегментация текста на отдельные слова с использованием модели **DBNet++ (ResNet-50)** от Mindee. Алгоритм устойчив к наклону текста и шумам.
* **Распознавание рукописного ввода (TrOCR)**: Преобразование изображений слов в текст с помощью трансформера **TrOCR** (`kazars24/trocr-base-handwritten-ru`), предобученного на русскоязычном датасете HKR.
* **Автоматическое исправление ошибок**: Интеграция с **Yandex Speller API** для проверки орфографии и исправления опечаток с учетом контекста предложений.
* **Интерфейс Gradio**: Веб-интерфейс, оптимизированный для загрузки файлов изображений, с отображением исходного текста, скорректированного текста и визуальной разметки сегментов.
* **Гибридный запуск**: Автоматическое использование видеокарты (NVIDIA CUDA) для высокой скорости инференса или переключение на процессор (CPU), если GPU недоступен.
